## Quick Setup

### System Requirements

- Bun 1.0+
- Node.js 18+
- Stripe Account

```bash
# Clone the repository
git clone https://github.com/duggal1/stripe-kit.git

# Navigate to the project directory
cd stripe-kit

# Install dependencies
bun install

# Launch the development server
bun dev
```
